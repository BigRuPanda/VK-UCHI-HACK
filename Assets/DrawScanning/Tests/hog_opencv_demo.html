🔬 DrawScanning — HOG via OpenCV.js

Сравнивает HOG-дескрипторы эталона и рисунка. Score = cosine similarity.
Открывать напрямую в браузере — сервер не нужен. OpenCV.js загружается с CDN (~8 MB).

Алгоритм: Resize → 128×96 → Grayscale → CLAHE (выравнивание гистограммы) → cv.HOGDescriptor.compute() → Cosine similarity.
Параметры HOG: winSize=128×96, blockSize=16×16, blockStride=8×8, cellSize=8×8, nbins=9 → вектор ~1764 float.
