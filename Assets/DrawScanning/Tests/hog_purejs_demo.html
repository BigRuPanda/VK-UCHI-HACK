🟢 DrawScanning — HOG Pure JS (no dependencies)

Сравнивает HOG-дескрипторы без OpenCV.js. Score = cosine similarity.
Открывать напрямую в браузере — сервер не нужен. Никаких зависимостей.

Алгоритм (чистый JS, ~150 строк): Resize → 128×96 → Grayscale → Gaussian blur 5×5 → Sobel 3×3 (Gx, Gy) → HOG: 8×8 cells, 9 bins, L2-norm per 2×2 block → Cosine similarity.
Зависимости: нет — только Canvas 2D API (встроен в браузер). Подходит для production в Unity WebGL без загрузки OpenCV.js.
